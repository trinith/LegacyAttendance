# LegacyAttendance
A WoW addon to help with attendance tracking.

# Installation
Download the code as a zip file and extract the `LegacyAttendance` folder (the one inside `LegacyAttendance-main`) into your appropriate WoW `Interface\Addons` folder (ie, `<wow dir>\_classic_era_\Interface\AddOns`).

# Usage
* You can use `/LegacyAttendance` or `/la` to activate the addon.
* Enter `/la help` for in-game usage.

# Quick Start Guide
**IMPORTANT NOTE:** You must have permissions to see officer note in order to take attendance. The main names for ranks other than Raider and Officer are stored there.

1. Type `/la log` to capture the current raid attendance. This can be done multiple times and new players will be appended to the list.
2. Type `/la get today` to get the record for today.
3. Using your favourite chat copy/paste addon, copy the output and paste it into the attendance spreadsheet (WIP).
